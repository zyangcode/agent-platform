# 阶段 4 验收总结

## 1. 当前结论

阶段 4 Agent Team 已完成代码级收口，当前状态为：可进入浏览器手工验收。

本阶段完成的是最小 Team Demo 闭环，不是完整工作流平台：

```text
Profile executionMode=TEAM
  -> Chat /api/chat/stream
  -> Planner 生成 TaskPlan
  -> Executor 执行 TOOL_TASK / MODEL_TASK
  -> Orchestrator 生成 answerDraft
  -> Reviewer 审查
  -> Orchestrator 输出 final answer
  -> Gateway/Web SSE 实时透传 team_* 事件
  -> Frontend Chat / Trace 展示 Team 过程
```

## 2. 已完成能力

- Profile 支持 `BASIC` / `TEAM` 执行模式。
- 普通 User 可创建自己的 private Team Profile。
- Team 不是独立入口，仍复用现有 Chat 入口。
- AgentTool 统一 Skill/MCP 工具视图。
- Planner 支持 JSON 规划、修正重试和通用 `MODEL_TASK` fallback。
- Executor 支持授权工具白名单校验、工具任务和模型任务。
- Reviewer 支持审查结果、重试建议和失败 fallback warning。
- Team Runtime 串起 Planner -> Executor -> Reviewer -> Final。
- TeamRunLimiter 限制任务数、重试次数、工具调用数、模型调用数和超时。
- Team SSE 事件实时输出到 Gateway/Web/Frontend。
- Chat 右侧展示 Team run 面板和 Mermaid 文本。
- Trace Detail 展示 Team span 友好名称和关键属性。
- Planner / Executor / Reviewer 模型调用 usage 汇总并进入 Token Usage 链路。
- BASIC 单 Agent 模式保留，不被 Team 替换。
